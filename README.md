# Jiaocai-Downlaoder
免登录下载国家中小学智慧教育平台上的PDF格式电子教材  

![demo](https://github.com/ZSCMRMZ/Jiaocai-Downlaoder/blob/main/demo.png)  

适配目前所有学段：**小学、小学（五·四学制）、初中、初中（五·四学制）、高中**  

# 未来更新功能
- [ ] 顶部Menu栏快捷打开教材页面
- [ ] 批量保存
- [ ] 多线程下载
